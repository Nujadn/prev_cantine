## Installation

Pour installer et exécuter ce projet sur votre machine locale, suivez les étapes suivantes :

1. Cloner ce dépôt :
   ```bash

   cd votre-nom-repo
    ```

2. Créer un environnement virtuel :
   ```bash
    python -m venv venv

    ```
